# REL06-BP02: Define and calculate metrics

## Description

Define SLIs và SLOs để measure reliability.

## Implementation Guidance

- Define key SLIs (latency, error rate, throughput)
- Set SLOs based on business requirements
- Calculate error budgets
- Use standard metrics (RED, USE)
- Review and adjust targets regularly

## Risk Level

Medium - Without SLOs, reliability is subjective.
