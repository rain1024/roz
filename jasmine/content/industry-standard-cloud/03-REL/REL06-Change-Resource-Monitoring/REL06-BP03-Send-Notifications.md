# REL06-BP03: Send notifications

## Description

Configure alerting để notify teams of issues.

## Implementation Guidance

- Define alert thresholds based on SLOs
- Implement tiered alerting (warning, critical)
- Route alerts to appropriate teams
- Avoid alert fatigue with meaningful alerts
- Include runbook links in alerts

## Risk Level

High - Without alerts, issues go undetected.
