# REL03-BP01: Choose how to segment your workload

## Description

Quyết định cách chia workload (microservices vs monolith).

## Implementation Guidance

- Evaluate team structure and capabilities
- Consider deployment frequency requirements
- Assess scalability needs per component
- Start simple, refactor when needed
- Use bounded contexts from DDD

## Risk Level

Medium - Wrong segmentation leads to distributed monolith.
