# REL09-BP04: Implement appropriate backup strategies

## Description

Choose backup strategy phù hợp với RPO/RTO.

## Implementation Guidance

- Use full, incremental, or differential backups
- Store backups in separate location
- Encrypt backups at rest
- Implement retention policies
- Consider backup storage costs

## Risk Level

Medium - Wrong strategy may not meet objectives.
