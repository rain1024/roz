# REL07-BP04: Load test your workload

## Description

Perform load testing để understand capacity limits.

## Implementation Guidance

- Define realistic load scenarios
- Test beyond expected peak load
- Identify breaking points
- Validate scaling behavior
- Document capacity limits

## Risk Level

High - Unknown capacity limits lead to outages.
