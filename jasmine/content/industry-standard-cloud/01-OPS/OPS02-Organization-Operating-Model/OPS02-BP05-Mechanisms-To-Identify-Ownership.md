# OPS02-BP05: Mechanisms exist to identify responsibility and ownership

## Description

Có cơ chế để xác định trách nhiệm (RACI matrix, service ownership).

## Implementation Guidance

- Implement RACI matrix for key decisions
- Service catalog with ownership information
- Tagging strategy for cloud resources
- Documentation standards include ownership
- Self-service ownership lookup tools

## Risk Level

Low - Without mechanisms, ownership becomes tribal knowledge.
