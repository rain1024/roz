# OPS05-BP01: Use version control

## Description

Sử dụng version control cho tất cả code và configuration.

## Implementation Guidance

- Git for all source code
- Infrastructure as Code in version control
- Configuration files versioned
- Branching strategy defined (trunk-based or GitFlow)
- Code review requirements enforced

## Risk Level

High - Without version control, changes are untracked and irreversible.
