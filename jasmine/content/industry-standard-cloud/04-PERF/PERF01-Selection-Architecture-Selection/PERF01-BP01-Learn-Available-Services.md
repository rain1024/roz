# PERF01-BP01: Learn about available services and resources

## Description

Understand các options có sẵn trước khi select resources.

## Implementation Guidance

- Research cloud provider offerings
- Understand service capabilities and limits
- Compare managed vs self-hosted options
- Stay updated on new services
- Build internal knowledge base

## Risk Level

Medium - Limited knowledge leads to suboptimal choices.
