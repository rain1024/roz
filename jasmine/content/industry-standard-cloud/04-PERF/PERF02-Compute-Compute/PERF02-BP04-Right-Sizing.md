# PERF02-BP04: Determine required configuration by right-sizing

## Description

Right-size instances based on actual usage.

## Implementation Guidance

- Analyze current utilization
- Identify over/under-provisioned instances
- Use right-sizing recommendations
- Test before changing production
- Review regularly (monthly/quarterly)

## Risk Level

Medium - Over-provisioning wastes money, under-provisioning hurts performance.
