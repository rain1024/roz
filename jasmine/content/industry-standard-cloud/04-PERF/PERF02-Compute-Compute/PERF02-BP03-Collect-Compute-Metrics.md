# PERF02-BP03: Collect compute-related metrics

## Description

Monitor CPU, memory, network metrics cho compute.

## Implementation Guidance

- Collect CPU utilization
- Monitor memory usage
- Track network I/O
- Measure disk I/O
- Set up performance dashboards

## Risk Level

Medium - Without metrics, issues go undetected.
