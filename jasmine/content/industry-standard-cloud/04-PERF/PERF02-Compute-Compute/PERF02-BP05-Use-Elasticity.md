# PERF02-BP05: Use available elasticity

## Description

Implement auto-scaling để match demand.

## Implementation Guidance

- Configure horizontal auto-scaling
- Set appropriate scaling thresholds
- Define min/max capacity
- Test scaling behavior
- Monitor scaling events

## Risk Level

High - Without elasticity, traffic spikes cause issues.
