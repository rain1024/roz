# PERF03-BP04: Select appropriate storage options

## Description

Choose storage tiering strategy.

## Implementation Guidance

- Use SSD for hot data
- Use HDD for cold data
- Implement automatic tiering
- Consider object storage for archives
- Use caching for frequently accessed data

## Risk Level

Low - Proper tiering optimizes cost-performance.
