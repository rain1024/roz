# COST02-BP06: Track project lifecycle

## Description

Manage resource lifecycle để avoid orphaned resources.

## Implementation Guidance

- Track resource creation dates
- Define resource expiration policies
- Clean up unused resources
- Archive old data
- Review regularly

## Risk Level

Medium - Orphaned resources waste money.
