# COST01-BP06: Monitor cost proactively

## Description

Proactive cost monitoring để catch issues early.

## Implementation Guidance

- Set up real-time cost monitoring
- Configure anomaly detection
- Alert on unusual spending
- Track cost trends
- Review daily/weekly

## Risk Level

High - Late detection increases waste.
