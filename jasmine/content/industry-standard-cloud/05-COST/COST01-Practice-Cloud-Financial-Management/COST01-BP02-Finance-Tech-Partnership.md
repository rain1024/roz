# COST01-BP02: Establish partnership between finance and technology

## Description

Foster collaboration giữa finance và engineering teams.

## Implementation Guidance

- Hold regular cost review meetings
- Share cost data transparently
- Align on cost terminology
- Create joint optimization initiatives
- Define shared cost metrics

## Risk Level

Low - Silos prevent effective cost management.
