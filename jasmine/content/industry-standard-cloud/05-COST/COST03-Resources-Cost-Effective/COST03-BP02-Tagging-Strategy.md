# COST03-BP02: Add organization information to cost and usage

## Description

Implement tagging strategy cho cost allocation.

## Implementation Guidance

- Define required tags
- Enforce tagging policies
- Include environment, team, project tags
- Automate tag compliance
- Report on untagged resources

## Risk Level

Medium - Without tags, costs cannot be attributed.
