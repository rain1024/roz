# SUS02-BP03: Stop resources no longer required

## Description

Clean up unused resources để reduce waste.

## Implementation Guidance

- Identify idle resources
- Implement automatic shutdown
- Schedule non-production shutdowns
- Review and delete unused resources
- Track cleanup savings

## Risk Level

Low - Unused resources waste energy and money.
